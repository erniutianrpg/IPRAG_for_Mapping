The `MessageUtil` class provides static utility methods to display different types of messages (error, info, success) in a JavaServer Faces (JSF) web application. Each method creates a `FacesMessage` with the specified message text and severity level, then adds it to the current `FacesContext` for display. The methods are: `showError()` for errors, `showInfo()` for informational messages, and `showSuccess()` for success notifications.
