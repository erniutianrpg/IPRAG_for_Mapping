This code defines a `JoinMeeting` request model for a BigBlueButton API. It handles parameters needed to join a meeting (meetingID, userID, fullName, password, etc.), validates them using constraints, and converts string parameters to appropriate types (boolean, long). It extends `RequestWithChecksum` for security and implements methods to populate parameters from a map and convert string values. The class ensures required fields are present and valid before processing the join meeting request.
