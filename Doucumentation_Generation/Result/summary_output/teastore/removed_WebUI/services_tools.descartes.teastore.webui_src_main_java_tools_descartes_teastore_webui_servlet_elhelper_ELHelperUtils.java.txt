This code provides utility functions for Java Server Pages (JSP) in a web application. It contains two main helper methods: `formatToPrettyDate()` converts ISO-formatted date strings to a more readable format ("yyyy MM dd - HH:mm:ss"), and `formatPriceInCents()` formats price values in cents into dollar strings (e.g., "$ 12.34"). The class is a singleton (`UTILS`) to ensure a single instance is used throughout the application.
