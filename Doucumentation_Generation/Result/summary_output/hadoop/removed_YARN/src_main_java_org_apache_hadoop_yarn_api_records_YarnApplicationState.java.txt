This code defines an enum `YarnApplicationState` that represents the possible states of an ApplicationMaster in Apache Hadoop YARN. The states include: `NEW` (just created), `SUBMITTED` (submitted), `RUNNING` (currently executing), `FINISHED` (completed successfully), `FAILED` (ended with errors), and `KILLED` (terminated by a user/admin). The enum is marked as stable and public for use in the YARN API. (Words: 59)
