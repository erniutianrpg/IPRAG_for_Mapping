The `CompositeIdFetcher` class is a unified identifier-based fetcher for bibliographic entries. It detects the type of identifier (DOI, arXiv, or ISBN) from a given string and delegates the search to the appropriate fetcher (`DoiFetcher`, `ArXivFetcher`, or `IsbnFetcher`). If the identifier is valid, it returns the corresponding `BibEntry`; otherwise, it returns an empty `Optional`. The class uses `ImportFormatPreferences` for formatting and supports retry mechanisms for ISBN searches. Currently, IACR identifier support is disabled pending rework.
