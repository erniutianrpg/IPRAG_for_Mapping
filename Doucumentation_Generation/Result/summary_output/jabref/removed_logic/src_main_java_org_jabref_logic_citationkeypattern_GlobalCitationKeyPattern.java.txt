The `GlobalCitationKeyPattern` class manages global citation key patterns for bibliographic entries. It extends `AbstractCitationKeyPattern` and initializes with a default pattern list. The `fromPattern` method creates an instance by splitting a string pattern. The `getLastLevelCitationKeyPattern` method returns the default pattern regardless of entry type, ensuring consistent citation key generation across all entry types. This class is used for maintaining uniform citation key patterns in a bibliography system. (94 words)
