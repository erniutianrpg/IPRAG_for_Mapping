The `IdentifierParser` class parses various identifier fields (DOI, ISBN, eprint, MR number) from a `BibEntry` in a BibTeX file. It checks the field type and uses appropriate parsers (like `DOI.parse()` or `ISBN.parse()`) to convert the field value into corresponding identifier objects. For eprint fields, it additionally checks the eprint type (e.g., "arxiv" or "ark") to determine the correct parser. Returns an `Optional` containing the parsed identifier or empty if parsing fails or the field is blank.
