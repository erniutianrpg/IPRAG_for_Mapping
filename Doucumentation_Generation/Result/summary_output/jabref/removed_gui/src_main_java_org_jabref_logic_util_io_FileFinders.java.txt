The `FileFinders` class provides a static method `constructFromConfiguration` that creates a `FileFinder` instance based on `AutoLinkPreferences`. Depending on the citation key dependency type (START, EXACT, or REGEX), it returns either a `CitationKeyBasedFileFinder` (with exact match or not) or a `RegExpBasedFileFinder` configured with a regex pattern and separator. This allows flexible file finding based on different citation key matching strategies. (98 words)
