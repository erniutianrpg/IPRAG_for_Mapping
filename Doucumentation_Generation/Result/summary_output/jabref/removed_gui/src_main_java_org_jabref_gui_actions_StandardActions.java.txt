The `StandardActions` enum defines all possible actions in the JabRef application, including their text labels, descriptions, icons, and keyboard shortcuts. It covers operations like copy/paste, file management, entry editing, library handling, search functions, and help resources. Each action is configured with optional icons and key bindings, providing a centralized repository of available commands for the GUI. This enum implements the `Action` interface, standardizing how actions are accessed and displayed throughout the application.
