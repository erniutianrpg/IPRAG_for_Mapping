The `OptionalObjectProperty<T>` class extends `SimpleObjectProperty<Optional<T>>` to provide enhanced JavaFX property bindings for `Optional` values. It includes methods to create an empty property (`empty()`), bind to a fallback value when empty (`orElseOpt(T)`), and check if a value is present (`isPresent()`). This simplifies working with `Optional` in JavaFX observables by enabling reactive operations on potentially empty values.
