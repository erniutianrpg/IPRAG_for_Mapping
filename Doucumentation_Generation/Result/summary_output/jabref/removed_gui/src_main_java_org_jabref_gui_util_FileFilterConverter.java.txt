This utility class `FileFilterConverter` provides methods to convert between different file filter types (JavaFX `ExtensionFilter`, Java `FileFilter`, and NIO `DirectoryStream.Filter`) for use in file dialogs. It supports creating filters for importers/exporters, converting file extensions, and handling file type descriptions. Key functionalities include converting `FileType` to extension filters, matching filters to importers/exporters, and creating combined filters for multiple formats. The class also handles localization of filter descriptions.
