The `ErrorConsoleViewModel` class manages error logging functionality in a JavaFX application. It provides methods to: 
1. Display and track log messages as observable data
2. Copy logs to clipboard 
3. Clear logs 
4. Report issues to GitHub by auto-generating a bug report template with system info and logs, then opening a browser. 

It integrates with the application's clipboard, dialog services, and logging system to handle error messages and facilitate issue reporting.
