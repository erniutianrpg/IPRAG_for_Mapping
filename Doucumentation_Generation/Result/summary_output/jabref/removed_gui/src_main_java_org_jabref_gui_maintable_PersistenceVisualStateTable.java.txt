The `PersistenceVisualStateTable` class tracks and persists visual state changes (column order, width, sorting) in a JavaFX TableView for a BibTeX entry table. It listens for column modifications and sorting changes, then updates the `ColumnPreferences` accordingly. The `updateColumns()` method saves column configurations (visibility, width, sort type), while `updateSortOrder()` specifically persists the current sort order. These preferences help maintain the table's visual state across sessions.
