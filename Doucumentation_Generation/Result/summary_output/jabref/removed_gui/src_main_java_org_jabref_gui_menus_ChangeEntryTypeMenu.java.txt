The `ChangeEntryTypeMenu` class creates a context or sub-menu for changing the type of selected BibTeX/BibLaTeX entries. It dynamically generates menu items based on the database mode (BibTeX or BibLaTeX), including default types, IEEETran types (for BibTeX), and custom types. The menu items trigger a `ChangeEntryTypeAction` to update the entry type. The class supports undo functionality via an `UndoManager`.
