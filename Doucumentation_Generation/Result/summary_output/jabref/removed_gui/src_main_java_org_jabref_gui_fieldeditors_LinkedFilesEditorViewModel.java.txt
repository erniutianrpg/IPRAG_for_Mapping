The `LinkedFilesEditorViewModel` class manages linked files for BibTeX entries in JabRef. It handles file operations like adding, downloading, and deleting files (both local and online), auto-linking associated files, and converting between file paths and linked file representations. The class integrates with JabRef's preferences, database context, and UI components to provide file management functionality while maintaining synchronization between the model and view. Key features include full-text fetching, URL-based file downloads, and automatic file detection.
