The `BibEntryTableViewModel` class manages and provides observable data for a BibTeX entry in a table view. It handles field values, special fields, linked files, identifiers, and matched groups. The class binds to entry fields, formats values, tracks changes, and provides observable properties for UI updates. Key functionalities include parsing linked files, combining identifiers (URL, DOI, etc.), matching groups, and managing special field values with proper synchronization between the model and UI.
