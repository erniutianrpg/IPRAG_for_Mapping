This JavaFX class `BibTexStringAddDetailsView` displays details of a newly added BibTeX string in a collab view. It shows a header "Added string" followed by the string's label and content in a vertical layout (VBox) with 8-pixel padding. The class extends `DatabaseChangeDetailsView` and is part of a collaboration feature in JabRef for tracking changes to bibliography strings. The displayed information is localized using `Localization.lang()`.
