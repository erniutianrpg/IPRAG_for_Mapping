The `StringUtil` class provides utility methods for string manipulation, including operations like quoting/unquoting, wrapping, trimming, case conversion, encoding/decoding, special character handling, and line break normalization. It supports functionalities such as joining strings, checking for whitespace, capitalizing text, replacing special characters, and handling various bracket types. The class also includes methods for HTML escaping, XML character validation, and efficient string-to-integer conversion. Many methods are optimized for performance and handle edge cases like null or empty strings.
