The `OutputPrinter` interface defines methods for displaying messages and status updates. It includes:
1. `setStatus(String s)` - Sets a status message
2. `showMessage(String message, String title, int msgType)` - Shows a message with title and type
3. `showMessage(String string)` - Shows a simple message

This interface provides a standard way for different classes to output information to users, supporting both simple messages and more detailed notifications with titles and message types.
