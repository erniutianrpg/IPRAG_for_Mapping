The `StringUtil` class provides utility methods for string manipulation, including:

1. **String Formatting**: Quote/unquote special characters, wrap text, capitalize first letters, strip accents, and normalize line breaks.
2. **Bracket Handling**: Check/add/remove curly/square brackets or citation marks around strings.
3. **Validation**: Check for null/empty strings, whitespace, or valid XML characters.
4. **Conversion**: Convert booleans to binary strings, strings to integers, and replace special characters with readable equivalents.
5. **Tokenization**: Split strings into words or sentences based on delimiters.
6. **File Handling**: Ensure correct file extensions and limit string lengths.

This class is part of the JabRef project and leverages Apache Commons Lang and Guava for some operations. It focuses on BibTeX-compatible string processing.
