This Java class `CreateMeeting` represents a request to create a meeting in a BigBlueButton (BBB) API. It validates and stores meeting parameters like name, ID, voice bridge, passwords (attendee/moderator), breakout room flag, and recording flag. The class includes constraints for parameter validation, methods to populate parameters from a map, and conversion of string parameters to appropriate types (integer/boolean). It extends `RequestWithChecksum` for security verification.
