This Java class `User` represents a participant in a BigBlueButton conference. It stores user attributes like ID, name, role (moderator/viewer), status (locked, guest, presenter), media streams (video/audio), and connection state (joined/left). The class provides methods to manage user properties, status flags, media streams, and track participation time. It's thread-safe, using concurrent collections for status and streams. Key functionalities include checking user roles, managing media streams, and tracking user presence.
