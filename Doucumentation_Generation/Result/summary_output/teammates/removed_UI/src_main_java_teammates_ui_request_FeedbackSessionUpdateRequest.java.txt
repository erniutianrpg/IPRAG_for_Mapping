The `FeedbackSessionUpdateRequest` class extends `FeedbackSessionBasicRequest` to handle updates for feedback sessions. It stores and manages student and instructor deadlines as maps of identifiers to epoch milliseconds. The class provides methods to convert these milliseconds into `Instant` objects for both student and instructor deadlines, along with setter methods to update these deadlines. Essentially, it facilitates deadline management for feedback sessions in a structured format. (94 words)
