This code defines a `FeedbackRankOptionsQuestionDetails` class for handling rank-options type feedback questions. It validates question details (ensuring proper option counts, non-empty options, valid min/max ranking constraints) and responses (checking for duplicate/invalid ranks, min/max ranking compliance). The class inherits from `FeedbackRankQuestionDetails` and includes methods to manage options list, detect changes requiring response deletion, and validate question/rules. Error messages are defined for various validation failures.
