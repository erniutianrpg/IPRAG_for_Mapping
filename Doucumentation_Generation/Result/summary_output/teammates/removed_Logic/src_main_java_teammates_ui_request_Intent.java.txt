This code defines an `Intent` enum that represents different purposes for calling a REST API in a feedback system. It includes five intents: `FULL_DETAIL` (for full entity details), `INSTRUCTOR_SUBMISSION` and `STUDENT_SUBMISSION` (for submitting feedback as instructors or students), and `INSTRUCTOR_RESULT` and `STUDENT_RESULT` (for viewing feedback results as instructors or students).
