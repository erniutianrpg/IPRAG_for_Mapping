The `DataBundleRegenerator` class regenerates JSON data files in specified directories to ensure consistent formatting. It processes three types of files: 1) DataBundle JSON files (excluding blacklisted files), 2) a feedback session template file, and 3) generic website data JSON files. For each file, it reads the content, converts it to a standardized JSON format (replacing timezone indicators), and saves it back. The script runs on files in `src/main/resources`, `src/test/resources/data`, and `src/main/webapp/data` directories.
