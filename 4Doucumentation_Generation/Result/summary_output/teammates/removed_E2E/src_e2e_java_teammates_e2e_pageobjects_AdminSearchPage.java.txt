This code defines the `AdminSearchPage` class, a Selenium Page Object for testing an admin search page in a web application. It provides methods to:

1. Search and manage student, instructor, and account request records
2. View and verify details in search results
3. Regenerate access keys for students/instructors
4. Reset Google IDs
5. Expand/collapse additional information links
6. Verify content and links in expanded rows
7. Delete or reset account requests

The class contains locators for page elements and helper methods to interact with and validate search results tables.
