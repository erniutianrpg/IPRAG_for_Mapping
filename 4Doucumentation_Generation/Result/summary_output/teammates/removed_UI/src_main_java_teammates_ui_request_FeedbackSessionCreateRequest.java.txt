This Java class `FeedbackSessionCreateRequest` extends `FeedbackSessionBasicRequest` and handles the creation of a feedback session. It includes fields for the session name (`feedbackSessionName`) and optional fields (`toCopyCourseId`, `toCopySessionName`) to copy an existing session. The class provides getters and setters for these fields and overrides the `validate()` method to ensure the session name is not null or empty, and that if a course ID is provided for copying, the corresponding session name is also provided. Validation throws an `InvalidHttpRequestBodyException` if checks fail.
