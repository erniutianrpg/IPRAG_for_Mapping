This Java enum `FeedbackResultFetchType` defines three options for fetching feedback response results: by giver only (`GIVER`), receiver only (`RECEIVER`), or both (`BOTH`). Each option has boolean flags indicating whether it fetches by giver or receiver. The `parseFetchType` method converts a string input (case-insensitive) into the corresponding enum value, defaulting to `BOTH` if invalid or null. Helper methods `shouldFetchByGiver()` and `shouldFetchByReceiver()` check the fetch type's flags. This enum helps specify how feedback results should be retrieved in the system. (98 words)
