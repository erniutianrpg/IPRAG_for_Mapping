The `StudentAttributes` class is a data transfer object (DTO) representing student information in a course. It encapsulates student details like name, email, course ID, team, section, comments, and Google ID. The class provides methods for validation, sanitization, sorting students by section/team, conversion to/from persistent entities, and builder patterns for creation/updates. It also handles registration status and generates registration URLs. The class ensures data integrity through validation and sanitization methods.
