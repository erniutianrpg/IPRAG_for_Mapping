The `SanitizationHelper` class provides utility methods to sanitize various types of user inputs to prevent security threats and ensure data consistency. It includes methods to sanitize Google IDs, emails, names, titles, and text fields by trimming whitespace. For rich text, it uses a predefined HTML policy to allow only specific elements and attributes. It also offers HTML escaping for safe insertion into HTML and URI encoding for URLs. All methods handle null inputs by returning null. The class is designed as a utility class with a private constructor.
