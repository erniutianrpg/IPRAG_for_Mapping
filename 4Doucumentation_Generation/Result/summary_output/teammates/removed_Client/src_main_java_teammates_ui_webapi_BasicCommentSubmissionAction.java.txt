This abstract class `BasicCommentSubmissionAction` extends `BasicFeedbackSubmissionAction` and provides common validation methods for feedback response comments in an educational system. It checks:  
1. If a question allows comments on responses  
2. If a comment already exists for a response  
3. If a student or instructor has ownership rights to modify a response  
Throws exceptions for invalid operations or unauthorized access. Serves as a base for comment-related actions. (98 words)
