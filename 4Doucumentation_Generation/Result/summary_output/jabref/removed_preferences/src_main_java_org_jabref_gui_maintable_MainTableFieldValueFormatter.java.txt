This code formats field values for a BibEntry table in JabRef, handling author names and LaTeX-to-Unicode conversion. The `MainTableFieldValueFormatter` class uses user preferences (display style and abbreviation style) to format fields, particularly focusing on author names. For non-name fields, it resolves fields with LaTeX expressions converted to Unicode. For name fields, it applies specified formatting (e.g., "First Last", "Last, First", or abbreviated forms) while ensuring LaTeX-free output. The main method `formatFieldsValues` processes fields and delegates name formatting to `formatFieldWithAuthorValue`.
