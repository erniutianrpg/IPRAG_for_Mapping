This utility class provides methods for validating meeting/presentation IDs, generating unique presentation IDs, creating/managing presentation directories, and handling downloadable markers. Key functions include:
- Validating ID formats using regex patterns
- Generating SHA1-hashed presentation IDs with timestamps
- Creating directory structures for presentations
- Managing downloadable status via marker files
- Path manipulation for presentation files

All methods are static, and the class is designed as a utility (non-instantiable) class.
