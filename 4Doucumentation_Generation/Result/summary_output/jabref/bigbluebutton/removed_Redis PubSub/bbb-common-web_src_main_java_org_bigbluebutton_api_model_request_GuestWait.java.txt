This code defines a `GuestWait` request class in a BigBlueButton API. It handles guest waiting functionality by validating: 1) user session via `sessionToken`, 2) meeting existence, and 3) meeting status. The class extracts the `sessionToken` from request parameters, sets it in a `SessionService`, and retrieves the associated `meetingID`. Key constraints ensure valid sessions and active meetings. The class implements the `Request` interface for parameter handling. (98 words)
