The `CreateMeetingMessage` class is a Java data structure representing a meeting creation request in BigBlueButton. It contains fields for meeting details like ID, name, recording settings, passwords, duration, camera limits, disabled features, metadata, and various configuration flags. The class serves as a container for all parameters needed to create a new meeting, initialized through its constructor.
