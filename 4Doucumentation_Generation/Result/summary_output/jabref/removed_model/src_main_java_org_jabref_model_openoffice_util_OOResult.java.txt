The `OOResult` class is a generic utility that represents either a successful computation result (`R`) or an error (`E`), but never both. It provides methods to check status (`isPresent()`, `isError()`), retrieve values (`get()`, `getError()`), apply functions conditionally (`ifPresent()`, `ifError()`), transform results (`map()`, `mapError()`), and convert to optional or void result. It ensures non-null values and enforces mutual exclusivity between result and error states.
