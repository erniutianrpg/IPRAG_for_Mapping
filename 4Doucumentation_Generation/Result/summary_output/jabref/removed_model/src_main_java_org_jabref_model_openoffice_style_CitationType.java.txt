This Java enum `CitationType` defines three citation presentation styles for academic references: `AUTHORYEAR_PAR` (author-year in parentheses), `AUTHORYEAR_INTEXT` (author-year in-text), and `INVISIBLE_CIT` (invisible citation). It provides two methods: `inParenthesis()` returns `true` for parenthetical or invisible citations, and `withText()` returns `false` only for invisible citations. Essentially, it helps manage how citations are displayed in documents. (98 words)
