The `OOVoidResult<E>` class represents a result that can either be successful (OK) or contain an error of type `E`. It provides methods to check the status (`isError`, `isOK`), retrieve the error (`getError`), handle errors (`ifError`), and transform errors (`mapError`). The class uses `Optional<E>` internally to store the error, ensuring non-null values. Static factory methods `ok()` and `error(E)` create instances for success and error cases, respectively. This is a utility class for functional error handling in void operations.
