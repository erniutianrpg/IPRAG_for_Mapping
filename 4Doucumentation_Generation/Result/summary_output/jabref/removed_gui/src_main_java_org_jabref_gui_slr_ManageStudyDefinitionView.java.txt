This JavaFX class `ManageStudyDefinitionView` provides a dialog interface for creating or editing a Systematic Literature Review (SLR) study definition. It allows users to:
1. Set study title and directory
2. Add/remove authors, research questions, and search queries
3. Select enabled databases
4. Save the study configuration

The UI includes tables for managing study components, input fields for adding new items, and validation to ensure required fields are filled before saving. The class supports both creating new studies and editing existing ones, with appropriate UI adjustments for each mode.
