The `OnlyIntegerFormatter` class is a JavaFX `TextFormatter` that restricts input to integers only. It uses an `IntegerStringConverter` for value conversion and an `IntegerFilter` that checks each input change against a regex pattern (`\\d*`) to allow only digits. If the input doesn't match (non-digit characters), it returns `null` to reject the change. The formatter can be initialized with a default value (or 0 if none provided). Essentially, it ensures text fields only accept valid integer input.
