This JavaFX ViewModel class manages the editing of linked files in a bibliography database. It handles:
1. Storing and updating file links (local or remote) and descriptions
2. Managing file type selections from available external file types
3. Providing file browsing functionality
4. Converting between absolute and relative paths
5. Creating new LinkedFile objects with proper metadata
The class interacts with the UI through JavaFX properties and handles file operations using preferences and database context.
