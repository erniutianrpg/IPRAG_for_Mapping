The `GroupTreeNodeViewModel` class is a view model wrapper for `GroupTreeNode` in JabRef, providing UI-related functionality for group management. It handles group descriptions, entry manipulation (add/remove), movement operations (up/down/left/right), child sorting, and undo operations. It supports different group types (Explicit, Keyword, Search, AllEntries) and provides dynamic formatting and validation checks for group operations. The class also manages subscriptions to group tree changes.
