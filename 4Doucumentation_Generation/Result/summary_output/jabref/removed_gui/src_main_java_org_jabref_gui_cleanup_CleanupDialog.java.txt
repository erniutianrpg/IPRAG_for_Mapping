The `CleanupDialog` class creates a JavaFX dialog for configuring entry cleanup preferences in JabRef. It displays a scrollable `CleanupPresetPanel` with OK/Cancel buttons. When OK is clicked, it returns the selected cleanup preferences; otherwise, it returns null. The dialog is sized 600x650 and titled "Cleanup entries". It uses the provided database context, initial cleanup preset, and file preferences to initialize the panel. (97 words)
