This code extends `AbstractUndoableEdit` to create a custom undo/redo functionality for JabRef. It overrides `getUndoPresentationName` and `getRedoPresentationName` to provide localized, HTML-formatted strings for undo/redo actions, combining a standard label ("Undo"/"Redo") with a context-specific action name from `getPresentationName()`. The class serves as a base for JabRef-specific undoable edits. (72 words)
