The `ReferenceViewModel` class represents a reference entry in a JavaFX UI, storing an entry key, highlight status, and associated citations. It provides methods to check if the entry is highlighted (`isHighlighted`), get a read-only list of citations (`getCitationList`), and display formatted text with the entry key and citation count (`getDisplayText`). The class encapsulates citation data for display purposes in a JabRef (reference management software) context.
