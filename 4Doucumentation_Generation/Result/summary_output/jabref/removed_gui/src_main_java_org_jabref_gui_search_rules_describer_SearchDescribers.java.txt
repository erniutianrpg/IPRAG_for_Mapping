This code provides a factory method `getSearchDescriberFor` that returns an appropriate `SearchDescriber` based on the type of search rule in a `SearchQuery`. It handles three types of search rules: `GrammarBasedSearchRule`, `ContainsBasedSearchRule`, and `RegexBasedSearchRule`, returning corresponding describers (`GrammarBasedSearchRuleDescriber` or `ContainsAndRegexBasedSearchRuleDescriber`) to make the search query human-readable. If the search rule type is unsupported, it throws an `IllegalStateException`.
