The `ISSNChecker` class validates ISSN (International Standard Serial Number) values. It checks if the input is blank, verifies the correct format (including hyphens), and validates the checksum digit. If the format is invalid, it returns an "incorrect format" error. If the checksum is wrong, it returns an "incorrect control digit" error. Valid ISSNs pass without errors. The checks are localized for user-friendly messages.
