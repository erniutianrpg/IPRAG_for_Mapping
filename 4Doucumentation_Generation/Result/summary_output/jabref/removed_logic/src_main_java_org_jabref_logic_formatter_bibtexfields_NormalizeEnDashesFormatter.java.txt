This Java class `NormalizeEnDashesFormatter` extends `Formatter` to normalize en dashes in BibTeX fields. It replaces single hyphens surrounded by spaces (" - ") with proper en dashes (" -- "). Key features include: a name/key for identification, a description, and an example input. The core functionality is in the `format` method, which performs the string replacement. This formatter is part of JabRef's logic for cleaning up BibTeX entries. (98 words)
