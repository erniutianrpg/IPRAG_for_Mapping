The `DateFormatter` class formats date strings from ISO format (`yyyy-MM-dd`) to a custom format specified via `setArgument()`. By default, it maintains ISO format. It implements `ParamLayoutFormatter`, parsing the input date using `ISO_LOCAL_DATE`, then reformats it using the provided pattern. The format pattern can be changed using `setArgument()`. For example, calling `setArgument("dd/MM/yyyy")` would output dates in day/month/year format.
