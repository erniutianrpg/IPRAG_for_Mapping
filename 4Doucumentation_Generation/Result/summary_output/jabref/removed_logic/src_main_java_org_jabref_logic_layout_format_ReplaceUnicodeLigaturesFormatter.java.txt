This code defines a `ReplaceUnicodeLigaturesFormatter` class that replaces Unicode ligatures (like "Æ") with their expanded forms (like "AE"). It uses a predefined map of ligature patterns (`UnicodeLigaturesMap`) to perform regex-based replacements on input text. The class implements `LayoutFormatter` and `Formatter` interfaces, providing metadata (name, key, description, example) and the core formatting functionality. For example, "Æneas" would be converted to "Aeneas".
