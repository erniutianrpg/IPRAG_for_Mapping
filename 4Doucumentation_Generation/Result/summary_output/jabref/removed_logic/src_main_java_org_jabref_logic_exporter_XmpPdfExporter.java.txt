The `XmpPdfExporter` class exports BibTeX entries as XMP metadata to PDF files. It checks if the target file is a PDF, then uses `XmpUtilWriter` to embed the entries' metadata into the PDF using XMP format. The exporter requires a database context, PDF file path, and entries list, throwing an exception if any are null. It's part of JabRef's logic for handling PDF metadata. (98 words)
