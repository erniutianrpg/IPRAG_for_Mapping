The `Meeting` class represents a meeting session in the BigBlueButton conferencing system. It stores meeting details like ID, name, passwords, participant limits, recording settings, and user management. Key functionalities include:

- Managing meeting properties (duration, recording, layout)
- Handling users (join/leave, roles, guest policies)
- Tracking breakout rooms and user activity
- Storing metadata and custom data
- Implementing builder pattern for meeting creation

The class provides methods to control meeting behavior, user access, and various meeting configurations through getters/setters. It's a core domain object that maintains the state of a meeting session.
