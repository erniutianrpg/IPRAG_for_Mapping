This Java interface defines a publisher service for a meeting system (likely BigBlueButton). It provides methods to manage meetings (create, destroy, end), handle user registration, send messages/keep-alives, and share STUN/TURN server info for WebRTC. Key functionalities include meeting lifecycle management, user management with roles, and real-time communication support through messaging and network info sharing. The interface supports features like recording, breakout rooms, and guest policies.
