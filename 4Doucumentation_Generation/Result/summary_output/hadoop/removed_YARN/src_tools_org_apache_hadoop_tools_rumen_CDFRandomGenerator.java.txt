This code defines an abstract class `CDFRandomGenerator` that generates random values conforming to a given discrete cumulative distribution function (CDF). It uses rankings and values from a `LoggedDiscreteCDF` to initialize lookup tables. The class provides methods to find floor indices in the CDF and retrieve corresponding values. Concrete subclasses must implement `valueAt()` to interpolate values based on a probability input. The main functionality is generating random values (`randomValue()`) using the CDF's distribution via a provided random number generator.
